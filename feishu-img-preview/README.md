# 飞书图片大图预览

**适用平台**：飞书 Web (`https://*.feishu.cn/*`)

## 功能

- **飞书文档**：按住 Alt 悬停图片，弹出大图浮窗（文档本身无预览功能）
- **多维表格**：按住 Alt 悬停图片，隐藏原生小预览、显示更大的浮窗

飞书电子表格（Canvas 渲染，无法访问图片元素）暂不支持。

## 用法

1. 安装脚本（Violentmonkey 追踪外部编辑）
2. 打开飞书文档或多维表格
3. 按住 `Alt`，鼠标移到图片上即弹窗；松开 `Alt` 关闭

## 说明

- 原创脚本，非改自第三方
- 飞书图片使用 `blob:` URL，同源鉴权，无需额外处理跨域
