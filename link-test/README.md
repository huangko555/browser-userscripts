# link-test —— 外部编辑同步链路测试脚本

## 这是什么
一个**自检用**的小脚本，用来验证「Claude 改文件 → Violentmonkey 自动同步 → 刷新网页就能看到效果」这条工作链路是否打通。

它不针对任何具体网站，在**任意网页**右下角注入一个蓝色小标签：

```
油猴脚本已注入 ✓ v0.3.0
```

标签里带一个版本号 (脚本顶部的 `VERSION` 常量)。

## 怎么用
1. 把本目录的 `link-test.user.js` 拖进 Violentmonkey 设置页 → 点 **「Track external edits / 追踪外部编辑」**(出现 "Stop tracking" 即为追踪中)。详见根目录 [`../README.md`](../README.md)。
2. 随便打开一个网页，确认右下角出现蓝色标签。
3. 改一下脚本里的 `VERSION`(例如 `0.3.0` → `0.3.1`) 并保存 → 回网页 **F5 刷新** → 看标签上的版本号有没有跟着变。
   - **变了** → 链路打通，以后改任何脚本都是「改文件 + 刷新」即可，无需复制粘贴。
   - **没变** → 追踪没生效，回去检查 Violentmonkey 里这个脚本是否处于 "Stop tracking" 状态 (说明正在追踪)。

## 注意事项
- `@match *://*/*`：作用于所有网页。验证完链路后，若不想每个页面都看到标签，可在 Violentmonkey 里**禁用**本脚本 (留着备用，不必删)。
- `@grant none`：不申请任何特殊权限，纯注入一个 DOM 元素，无副作用。
